
# Run the Outlaw Scanner on Your Phone

The phone version runs as a Streamlit web app. You open it in Safari or Chrome.

## 1. Create a GitHub repository

1. Sign in to GitHub.
2. Create a repository named `outlaw-mlb-scanner`.
3. Choose **Private** if you do not want the code publicly visible.
4. Upload every file in this folder.
5. Commit the files.

## 2. Deploy with Streamlit Community Cloud

1. Open Streamlit Community Cloud.
2. Sign in with GitHub.
3. Choose **Create app**.
4. Select your `outlaw-mlb-scanner` repository.
5. Set the main file path to `app.py`.
6. Deploy.

Streamlit will give you a web address.

## 3. Add it to your phone

### iPhone
1. Open the app address in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

### Android
1. Open the app address in Chrome.
2. Tap the three-dot menu.
3. Tap Add to Home screen or Install app.

## Daily use

1. Open the scanner.
2. Select the date.
3. Leave Morning roster scan on before lineups are confirmed.
4. Tap Run Full Scan.
5. Review Top 20, Core HR and Full board.
6. Download Excel or CSV.

## Notes

- The free host may sleep after inactivity.
- Public Statcast downloads can occasionally be slow.
- Final recommendations still require lineup, weather, scratches and price verification.
