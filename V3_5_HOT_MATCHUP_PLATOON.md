# V3.5 Hot Form, Best Matchup, and Platoon Markers

## Visual markers

- `🔥` = warm recent power/contact profile
- `🔥🔥` = very hot recent power/contact profile
- `🔥🔥🔥` = elite recent power/contact profile
- `*` = opposite-handed batter versus pitcher matchup
- `**` = switch hitter
- blank = same-handed matchup

## Best Matchup

The label combines:

- 40% pitch-mix compatibility
- 35% opposing-pitcher vulnerability
- 25% hitter contact quality

`BEST MATCHUP` is assigned to the highest-scoring top-six hitter within each
offense when the matchup score is at least 65.

These additions are display and filtering aids only. They do not alter the
existing V3.4 final model score.
